# Lacuna UI Reference Screenshots

Captured from the live Omarchy shell on DP-1 at 2560x1440.

## Captures

- `01-desktop-bar-overlays.png` - Current desktop with active Lacuna bar layout and persistent overlays.
- `02-lacuna-menu-default.png` - Default `lacuna.menu` sidebar.
- `03-lacuna-menu-settings-flyout.png` - Menu with attached Lacuna settings flyout.
- `04-lacuna-menu-shell-settings-flyout.png` - Menu with attached Omarchy shell settings flyout.
- `05-lacuna-menu-app-picker-flyout.png` - Menu with quick-launch app picker flyout.
- `06-lacuna-shell-settings-panel.png` - Standalone `lacuna.shell-settings` panel.
- `07-lacuna-reference-layout-extra-widgets.png` - Temporary reference layout enabling Lacuna widgets not present in the normal bar layout, plus `lacuna.desktop-clock`.
- `07-lacuna-reference-layout-geometry.json` - `omarchy-shell shell debugBarGeometry` output for the temporary reference layout.
- `08-lacuna-settings-persistence-panel.png` - Standalone `lacuna.settings-persistence` panel.
- `09-lacuna-design-overhaul-concept.png` - Proposed Lacuna shell surface overhaul concept.
- `09-lacuna-design-overhaul-concept.svg` - Editable source for the overhaul concept render.

## Notes

The temporary reference layout was written to `~/.config/omarchy/shell.json` only long enough to capture `07-lacuna-reference-layout-extra-widgets.png`, then the original shell config was restored.

Service-only plugins with no screenshot surface: `lacuna.state` and `lacuna.theme-preloader`.
