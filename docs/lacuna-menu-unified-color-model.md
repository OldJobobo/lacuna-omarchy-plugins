# Lacuna Menu Unified Color Model

Status: superseded

The unified color model is now part of the Lacuna Design Language. See
[`docs/lacuna-design-system/01-color.md`](lacuna-design-system/01-color.md),
section "The unified color model", for the canonical rule and rationale:
one theme accent for all non-destructive chrome, `danger` reserved for
destructive actions, via `toneAccent(tone)`.
